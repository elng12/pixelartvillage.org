# Pixel Art Village - 项目概要

## 项目简介
Pixel Art Village 是一个在线图像转像素艺术工具，支持18种语言的多语言网站。用户可以上传图片并转换为像素艺术风格，支持自定义调色板、抖动控制等功能。

## 核心功能
- 图像转像素艺术转换器
- 自定义调色板系统
- 实时预览功能
- 浏览器内安全处理（无需注册）
- 多语言支持（18种语言）
- 响应式设计

## 技术栈
- **前端**: React 18 + Vite
- **路由**: React Router v6
- **国际化**: react-i18next + i18next-http-backend
- **样式**: Tailwind CSS
- **构建**: Vite
- **测试**: Playwright

## 支持语言
英语(en)、西班牙语(es)、印尼语(id)、德语(de)、波兰语(pl)、意大利语(it)、葡萄牙语(pt)、法语(fr)、俄语(ru)、菲律宾语(fil)、越南语(vi)、日语(ja)、瑞典语(sv)、挪威语(no)、荷兰语(nl)、阿拉伯语(ar)、韩语(ko)、泰语(th)
