# Project Brief: pixelartvillage.org (v2)

## 1. 项目愿景
**一比一复刻** "pixelartvillage.com" 网站，实现其完整的视觉、功能和交互。

## 2. 核心功能模块 (基于逆向分析)

### 2.1 图像上传模块
- **交互**: 支持拖拽上传和点击按钮选择文件。
- **格式**: 支持 JPG, PNG, GIF 等常见图像格式。
- **反馈**: 上传后立即在预览区显示原图。

### 2.2 像素化参数设置模块
- **核心参数**:
    - Pixel Size (像素大小)
    - Brightness (亮度)
    - Contrast (对比度)
    - Saturation (饱和度)
- **高级功能**:
    - 颜色数量限制 (调色板量化)
    - 风格预设 (如 8-bit, 16-bit)

### 2.3 实时预览模块
- **布局**: 左右分栏，清晰对比原图和效果图。
- **交互**: 参数调整后，效果图实时渲染更新。

### 2.4 下载与分享模块
- **下载**: 支持将处理后的图像下载为 PNG 或 SVG 格式。

## 3. 技术实现方案

### 3.1 核心像素化逻辑
- **技术**: 使用 HTML5 Canvas API。
- **原理**:
    1. 将原图绘制到一个临时 Canvas 上。
    2. 将 Canvas 尺寸缩小到目标像素尺寸 (`width / pixelSize`)。
    3. 关闭图像平滑 (`ctx.imageSmoothingEnabled = false`)，将原图绘制到缩小的 Canvas 上，强制浏览器进行像素合并。
    4. 将 Canvas 尺寸放大回原尺寸。
    5. 将缩小的 Canvas 内容绘制到放大的 Canvas 上，形成清晰的块状像素效果。

### 3.2 前端技术栈
- **Web 框架**: Next.js
- **UI 框架**: React
- **样式方案**: Tailwind CSS

## 4. 复刻计划
1.  **项目初始化**: 搭建 Next.js + Tailwind CSS 项目结构。
2.  **静态布局**: 首先完成登陆页面和工具页面的静态 UI 布局。
3.  **核心功能**: 实现 Canvas 像素化核心逻辑。
4.  **交互完善**: 联通 UI 控件与 Canvas 逻辑，实现参数调整和实时预览。
5.  **收尾工作**: 实现下载功能并完善细节。
